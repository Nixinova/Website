MineoCS Version History

Version Image Date Changelog
Alpha 0.4.2 7 January 2021 Additions
  • Pause menu.
  • Level saving in three slots from the pause menu.
  • Level loading from the main menu.
  • World seeds, which control all aspects of world generation.
  • Snowy grass (grass_snowy, ID 11) which generates in spruce forests.
 zip
Alpha 0.4.1 6 January 2021 Additions
  • World type selection menu.
    • World types are normal and flat.
  • Loading message when generating a world.
Changes
  • Smoothened out the top level of the terrain.
  • Made the block preview partially transparent.
  • Improved building performance.
 zip
Alpha 0.4.0 5 January 2021 Additions
  • Placing and breaking blocks.
    • Left-click removes blocks, right-click places blocks.
    • Use the number keys or scroll wheel to switch between blocks to place.
    • Toggle the ability to build using key B.
    • A block selection outline will appear in the currently hovered-over block.
  • The hotbar, which displays available blocks.
  • Sandstone (ID 4), forest grass (grass_forest, ID 10), and sand (ID 12).
  • Re-added nuking (key N), which removes random blocks in a 5×5×5 area around the player.
  • Re-added the crosshair.
Changes
  • Renamed grass (ID 12) to grass_plains (ID 9).
  • Changed many block IDs:
    • stone 4→1, clay 5→2, oak_leaves 20→24, redwood_leaves 21→25, spruce_leaves 22→26, copper 24→32, asphalt 25→33, tin 26→34, debug 252→240.
Removals
  • Workbench and quarry.
  • Tree IDs.
 zip
Alpha 0.3.0 23 March 2019 Additions
  • Workbench.
    • Found inside the spawn building.
    • When clicked on, it will open a crafting menu.
  • Crafting inside a workbench.
    • A quarry can be crafted for 32 stone.
Changes
  • Quarry no longer destroy mined blocks, as it caused massive lagspikes.
  • Mined blocks now display as a number on-screen.
 zip
Alpha 0.2.4 17 March 2019 Additions
  • Blocks mined using the quarry now show up on-screen.
 zip
Alpha 0.2.3 10 March 2019 Changes
  • Quarry now mines blocks one by one.
    • Hold down the left mouse button to use the quarry.
    • Note that this is quite laggy.
 zip
Alpha 0.2.2 10 March 2019 Additions
  • Quarry, which spawns in a random location in the world.
    • When clicked it will destroy all blocks below it with a 1-block radius all the way down to bedrock.
Changes
  • Exiting to the main menu no longer resets the map.
 zip
Alpha 0.2.1 10 March 2019 Changes
  • Not all of the spaceship's roof will generate now.
  • Changed mouse lock angle to 90° up and down.
  • Added foundations to spaceship.
  • Moved camera down half a block.
Fixes
  • Fixed trees and grass generating inside ship.
 zip
Alpha 0.2.0 9 March 2019 Additions
  • A broken spaceship at spawn (0,0).
    • Will play in to the upcoming storyline.
    • Made of new blocks copper (ID 24), asphalt (25), and tin (26).
Changes
  • Sped up player by 25%.
  • Raised camera by half a block.
Fixes
  • Fixed coordinates not displaying the correct block.
 zip
Alpha 0.1.5 3 March 2019 Changes
  • Trees can now generate anywhere inside a chunk.
  • Trees now generate with dirt under them.
Fixes
  • Fixed camera clipping through blocks above the player.
  • Fixed floating trees.
 zip
Alpha 0.1.4 3 March 2019 Changes
  • Blocks now take up one full world grid space instead of one quarter of a block.
  • The player now takes up 1×2×1 blocks.
  • Coordinates can now be toggled by pressing B.
  • Rounded coordinates to 1 decimal place.
  • Changed view lock angle from 45° to 60° up and down.
  • Removed another 4 layers of stone to speed up load times.
 zip
Alpha 0.1.3 28 February 2019 Additions
  • Nuking by pressing N.
    • Destroys random blocks with a 1 in 3 chance around the player.
    • Maximum area of 8×8×8.
Changes
  • Removed 4 layers of stone to speed up load times.
  • Reduced startup time.
  • Reduced player speed.
Removals
  • Removed the crosshair.

    • Menu screen.

      • Press "Start" to generate a level, press "Exit" to exit the game.

    • Modified texture of all logs.
    • Removed leaves that stick out too far from the tree.
    • Rearranged block tile set to allow for new block variants.
    • Retextured birch log and leaf.
    • Reset FOV to 45.
    Removals
    • Texture debug1.
    • Birch log.

    Indev 0.0.6 19 February 2019 Additions
    • Bedrock (ID 0), which generates at y=0.
    • Spruce log and leaf (IDs 26 and 34), which have a darker texture than oak.
    • Spruce trees, which can be 6–8 blocks tall.
    • Chunks, which are set to a random size between 3 and 6.
    • Added two debug blocks.
    Changes
    • Birch trees now grow to 7–11 blocks tall and are thicker around the middle.
    • Trees now have a 1/17 chance of generating in any given chunk.
    • Trees now has an even chance of being one of the three types.
    • Changed camelCase block names to underscore_case:
      • logOak→oak_log, logBirch→birch_log, leafOak→oak_leaf, leafBirch→birch_leaf.
    • Changed many block IDs:
      • dirt 2→8, grass 3→16, oak_log 4→24, oak_leaf 5→32, birch_log 6→25, birch_leaf 7→33.
    • Smoothened terrain generation using chunks.
    • Changed down button from LCtrl back to LShift.
    • Tilting the camera no longer moves the player.
    • Rearranged block tile set to allow for new block variants.
    • Retextured birch log and leaf.
    Removals
    • Removed unused forest grass texture and air block ID.
    • Removed biomes, which are replaced by the better named chunks.
    		 zip
    Indev 0.0.5 17 February 2019 Additions
    • Precursor to biomes.
      • Plains and forest biomes.
      • Each block column (Y-level) has a 2/3 chance of being a plains biome and 1/3 chance of being a forest.
    • Oak and birch logs and leaves (IDs 4–7).
      • Leaves are semi-transparent
    • Oak and birch trees.
      • Have a 1/64 chance of generating in plains biome and 1/8 chance of generating in forest biome.
      • Oak trees generate in plains, birch trees generate in forests.
    • Unused forest grass texture.
    Changes
    • Modified world generation to be semi-layered.
      • Min Y-coordinate of grass is now randomised between the last grass Y value and the max grass value (16).
    		 zip

    • Changed down button from LShift to LCtrl.
    Fixes
