Legacy – Image to Bundle Converter

Converts an image into bundle art for Minecraft 1.17–1.20.

This tool no longer works in versions 1.21.2 and above!

GUI scale must be set to 1 to view these bundles in full.
Pictures must be roughly square to display properly.




Output    

Note: This command is too long to be put in chat so it must be run from a command block: /give @s minecraft:command_block 1

/give @p minecraft:bundle{}

* Resolutions above ~30 may create bundles with data too long to be placed into a command block, so it is recommended to use a datapack function for these.
